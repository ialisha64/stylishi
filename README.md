## ğŸ”§ Advanced Usage

### Use Your Own Dataset

Want to search your own fashion images? Easy:

1. Place images in `images_catalog/` folder
2. Run:
   ```bash
   python download_dataset.py --build-index-only
   ```
3. Launch app: `streamlit run app.py`

### Use Larger Datasets (50k+ images)

The script supports:
- **DeepFashion2** (50k items) - [Download here](https://github.com/switchablenorms/DeepFashion2)
- **Fashion Product Images** (44k items) - Kaggle dataset
- **Your custom dataset** - Any image folder

Instructions:
```bash
python download_dataset.py --large-dataset
```

### Enable GPU Acceleration

If you have NVIDIA GPU:

1. Install FAISS-GPU:
   ```bash
   pip uninstall faiss-cpu
   pip install faiss-gpu
   ```

2. The app auto-detects GPU and uses it for embeddings (~10x faster!)
